## 内容要求

### 质量标准

1. **数据准确性**
   - 所有数据需标注来源
   - 价格/功能信息需注明更新日期
   - 避免过时信息

2. **实用性**
   - 提供可操作的建议
   - 包含实际案例
   - 避免纯理论描述

3. **双语一致性**
   - EN/ZH 内容核心信息一致
   - 本地化而非直译
   - 例子考虑文化差异

4. **SEO 优化**
   - 标题包含关键词
   - Meta description 完整
   - 内部链接合理

### 格式规范

- 使用产品手册中的内容模板
- 遵循设计系统规范
- 表格使用统一样式
- 图片使用 WebP 格式
