## 检查清单

### A. 跳转与导航检查 (每页必检)

#### A1. 语言切换器
- [ ] EN 页面的"中文"链接 → 正确跳转到对应 ZH 页面
- [ ] ZH 页面的"English"链接 → 正确跳转到对应 EN 页面
- [ ] 无 EN 对应的 ZH 页面 → 链接到 ZH 首页或父级（合理回退）
- [ ] 切换器样式统一（导航栏内标准样式）

#### A2. 内部链接
- [ ] 导航栏所有链接可点击且正确
- [ ] 页面内目录/锚点链接正确
- [ ] 无 404 错误链接
- [ ] 无旧路径链接（应为 `/en/*` 或 `/zh/*`，非 `/knowledge-base/*` 等）

#### A3. Canonical / Hreflang
- [ ] Canonical URL 指向自身（非他页）
- [ ] Hreflang 配对正确（EN↔ZH 双向）
- [ ] 无配对页面应有合理回退

### B. 语言对应检查

#### B1. 内容纯净度
- [ ] EN 路径无中文残留（排除语言切换器）
- [ ] ZH 路径无英文污染（排除专有名词和语言切换器）
- [ ] 日期格式统一（EN: `Mar 16, 2026` / ZH: `2026 年 3 月 16 日`）
- [ ] 数字格式统一（EN: `1,234` / ZH: `1,234`）

#### B2. 内容对应
- [ ] EN/ZH 配对页面主题一致
- [ ] 关键信息无遗漏（一方有重要内容，另一方也应有）
- [ ] 图表/表格在双语中都有对应

### C. 视觉效果检查

#### C1. 双层视觉架构
- [ ] 概览层：深色背景（`#0a0a0a` 或类似）
- [ ] 内容层：浅色背景（`#ffffff` 或 `#f8f9fa`）
- [ ] 过渡清晰，无视觉混淆

#### C2. 组件统一
- [ ] 导航栏样式全站一致
- [ ] 按钮样式统一（颜色、圆角、悬停效果）
- [ ] 卡片样式统一（阴影、边框、间距）
- [ ] 字体大小/行高统一（正文 `16px` / 行高 `1.6`）

#### C3. 响应式布局
- [ ] 桌面端（1920px）布局正常
- [ ] 平板端（768px）布局正常
- [ ] 移动端（375px）布局正常
- [ ] 无内容溢出或重叠

#### C4. 颜色系统
- [ ] 主色调一致（参考设计系统）
- [ ] 链接颜色统一（未访问/已访问/悬停）
- [ ] 警告/错误颜色统一
- [ ] 无突兀的自定义颜色

### D. 功能检查

#### D1. 搜索功能（如有）
- [ ] 搜索框可输入
- [ ] 搜索结果正确
- [ ] 无搜索结果为空状态友好

#### D2. 表单（如有）
- [ ] 输入框可聚焦
- [ ] 验证逻辑正确
- [ ] 提交成功/失败反馈清晰

#### D3. 交互元素
- [ ] 所有按钮可点击
- [ ] 悬停效果正常
- [ ] 动画流畅（无卡顿）

### E. 性能检查

#### E1. 加载速度
- [ ] 首屏加载 < 3 秒
- [ ] 图片懒加载生效
- [ ] 无阻塞资源

#### E2. 资源优化
- [ ] 图片格式正确（优先 WebP）
- [ ] CSS/JS 无冗余
- [ ] 无 404 资源请求

### F. SEO 检查

#### F1. Meta 标签
- [ ] Title 标签唯一且描述性
- [ ] Meta description 存在且相关
- [ ] Open Graph 标签完整（og:title, og:description, og:image）

#### F2. 结构化数据
- [ ] JSON-LD 存在（如适用）
- [ ] Schema.org 标记正确

#### F3. Sitemap
- [ ] sitemap.xml 包含所有页面
- [ ] URL 路径正确（新结构）
- [ ] 无旧路径残留
