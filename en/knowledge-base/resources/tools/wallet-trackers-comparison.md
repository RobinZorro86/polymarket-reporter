### 5.2 Risk Management Strategies

**The 2% Rule**:
- Never risk more than 2% per trade
- Example: $1,000 account → $20 max per trade
- Protects against large losses

**Portfolio Allocation**:
```
Beginner (Total: $500):
- Copy Trader A: $100 (20%)
- Copy Trader B: $100 (20%)
- Manual trading: $300 (60%)

Intermediate (Total: $5,000):
- Copy Trader A: $500 (10%)
- Copy Trader B: $500 (10%)
- Copy Trader C: $500 (10%)
- Manual trading: $3,500 (70%)

Advanced (Total: $20,000):
- Copy 5 traders: $2,000 each (50% total)
- Manual trading: $10,000 (50%)
```

**Stop-Loss Strategies**:
1. **Fixed Stop**: Close copy if trader loses 20%
2. **Trailing Stop**: Move stop up as profits increase
3. **Time Stop**: Review after 30 days, pause if underperforming

**Automating Your Strategy**:

**Python Example**:
```python
import requests
import time

# Configuration
API_KEY = "your_api_key"
TRADER_ID = "trader_to_copy"
MAX_COPY_AMOUNT = 100  # USD

# Monitor trader
while True:
    # Get trader's latest trade
    response = requests.get(
        f"https://api.tradefox.ai/traders/{TRADER_ID}/latest",
        headers={"Authorization": f"Bearer {API_KEY}"}
    )
    
    trade = response.json()
    
    # Check if we should copy
    if trade['confidence'] > 0.7:
        # Execute copy trade
        copy_trade(trade, MAX_COPY_AMOUNT)
        print(f"Copied trade: {trade['market']}")
    
    # Wait before next check
    time.sleep(60)
```

## 7. Extended FAQ

**Q1: Best for beginners?**
A: Limitless - user-friendly, free tier, mobile apps.

**Q2: Are they safe?**
A: Yes, use read-only access, 2FA, start small.

**Q3: How much to allocate?**
A: 5-10% for beginners, max 30% for advanced.

**Q4: Multiple trackers?**
A: Yes, professionals use 2-3 for diversification.

**Q5: Trader losing money?**
A: Set stop-loss, monitor weekly, diversify.

**Q6: Technical skills needed?**
A: No for Limitless/OpinionLabs, yes for Simmer.

**Q7: How to verify trader?**
A: Check 3+ months history, win rate >55%, max drawdown <20%.

**Q8: What if platform goes down?**
A: Have backup manual tracking, use multiple platforms.

**Q9: Can I lose more than I invest?**
A: No, maximum loss is your copy amount per trade.

**Q10: How often should I review?**
A: Daily for first week, then weekly, monthly for established traders.

**Q11: What metrics matter most?**
A: Win rate, Sharpe ratio, max drawdown, consistency.

**Q12: Should I copy during volatile markets?**
A: Reduce copy amounts, increase monitoring frequency.
