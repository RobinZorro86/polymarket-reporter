👛 Wallet Setup Guide

Last updated: 2026-03-07 · Source: Polymarket + Simmer Docs
Security MetaMask Polygon

📌 Overview

Polymarket runs on the Polygon network. You'll need a Polygon-compatible wallet with USDC.e (bridged USDC) for trading and a small amount of POL for gas fees.

⏱️ Estimated time: 10-15 minutes

Install MetaMask

Browser Extension (Recommended)

  1. Open Chrome / Edge / Firefox browser
  2. Visit metamask.io
  3. Click "Download" to install the extension
  4. Create a new wallet or import an existing one

Mobile App

  • iOS: Search "MetaMask" in the App Store
  • Android: Search "MetaMask" in Google Play

⚠️ Security alert: Your seed phrase can only be backed up once. Write it down offline (pen and paper). Never screenshot it or store it in the cloud!

Add Polygon Network

MetaMask defaults to Ethereum mainnet. You need to add Polygon manually:

Auto-add (Recommended)

  1. Visit polygon.technology
  2. Click "Add Polygon to MetaMask"
  3. Confirm the addition in the MetaMask popup

Manual Configuration

Network Name: Polygon Mainnet
RPC URL: https://polygon-rpc.com
Chain ID: 137
Currency Symbol: POL
Block Explorer: https://polygonscan.com

Get USDC.e

Polymarket uses USDC.e (bridged USDC), not native USDC.

Option A: Withdraw from Exchange

  1. Buy USDC on Binance / Bybit / other exchange
  2. Select withdraw → Choose Polygon network
  3. Enter your MetaMask address (starts with 0x)
  4. Wait 5-10 minutes for confirmation

Option B: Use a Bridge

  1. Visit bridge.polygon.technology
  2. Connect your wallet
  3. Select Ethereum → Polygon
  4. Enter amount and confirm the bridge

💡 Pro tip: Start with $50-100 for your first transfer to test the flow before moving larger amounts.

Get POL for Gas

You'll need a small amount of POL (typically $5-10 worth) for transaction fees.

From Exchange

  • Buy POL on your exchange
  • Withdraw to your MetaMask address on Polygon network

Swap within MetaMask

  • Use MetaMask's built-in swap feature
  • Swap a small amount of USDC.e → POL

✅ Verification Checklist

  • ✅ MetaMask installed and wallet created
  • ✅ Polygon network added and selected
  • ✅ USDC.e balance visible (not native USDC)
  • ✅ POL balance for gas fees (at least $5 worth)
  • ✅ Seed phrase backed up offline

⚠️ Common Mistakes

  • Wrong network: Make sure you're on Polygon, not Ethereum mainnet
  • Wrong token: Polymarket needs USDC.e, not native USDC or USDT
  • Insufficient gas: Keep at least $5-10 POL for transactions
  • Lost seed phrase: Without it, you lose access to your wallet forever
