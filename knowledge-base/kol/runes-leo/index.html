🎯 什么是 Pre-Market

Polymarket 的 Pre-Market 是一类特殊市场，预测 Crypto 项目的未来事件。主要三类：

FDV 市场

"MegaETH 上线首日 FDV 会超过$2B 吗？"

Airdrop 市场

"Pump.fun 会在 3 月 31 日前空投吗？"

Token Launch 市场

"Base 会在 2026 年底前发币吗？"

📈 三大关键洞察

1. FDV 市场是 No 的金矿

  • 数据：298 个已结算 FDV 市场，72.8% 结果是 No
  • 原因：项目方在 Pre-Market 阶段总是被高估
  • 策略：集中火力在高阈值（$4B+）买 No

2. Airdrop 市场几乎一样好

  • 数据：73% 的 No 胜率
  • 原因：空投永远比预期晚（Monad 从 9 月延到 12 月）
  • 策略：只买近期截止的（30-90 天）

3. Token Launch 是陷阱

  • 数据：50/50 没有 alpha
  • 原因：发币是迟早的事，长期限市场（2026 年底前）几乎 100% 结算为 Yes
  • 策略：回避此类市场
✅ 结论：集中火力在 FDV 和 Airdrop，回避 Token Launch。

🪜 阶梯效应（Staircase Pattern）

FDV 市场有个特别漂亮的规律——阶梯效应：

原理

以 Monad 为例：上线首日 FDV 落在$3.2B。

  • 所有低于$3.2B 的阈值（$2B、$3B）→ 结算为 Yes
  • 所有高于$3.2B 的阈值（$4B、$5B、$6B...）→ 结算为 No

策略应用

你不需要精确预测 FDV 是多少，只需要判断"不太可能超过$X"：

Monad 案例：
同时买$5B No、$6B No、$8B No、$10B No
→ 四个全赢

⚙️ 策略框架

选品规则

  • ✅ FDV 市场：优先
  • ✅ Airdrop 市场：可选
  • ❌ Token Launch：回避

入场规则

  • FDV：选中间偏高阈值（如$3B-$4B 区间）
  • Airdrop：只买近期截止的（30-90 天）
  • 价格：No 价格低于 70c 才有吸引力（ROI > 43%）

仓位管理

  • 单个市场 ≤ 总仓位 10%
  • 同一事件（不同阈值）合计 ≤ 30%
  • 保持 5-10 个活跃持仓的分散度

退出规则

  • 核心策略：持有到结算（不做波段）
  • 例外：如果结算前 No 价格涨到 95c+，考虑提前卖出锁定利润

⚠️ 风险和陷阱

  • 黑天鹅事件：Plasma 上线 FDV 一度冲到$8B，所有阈值 No 归零（27% 的 FDV 市场结果是 Yes）
  • 时间成本：No @ 90c 的交易，赢了只赚 11%，锁仓 3 个月年化 44%
  • 流动性风险：只做交易量 >$100K 的市场
  • Token Launch 陷阱：2026 年到期的市场，93% 结算为 Yes
