📌 什么是 Simmer

Simmer 是 AI Agent 专用的预测市场交易接口，支持 Polymarket 和 Kalshi 两个平台。提供自托管钱包、安全限额和智能上下文建议。

💰 起始资金：注册即送 10,000 $SIM（虚拟币），可用于练习交易。
交易入门虚拟盘实盘

🚀 快速开始

1. 注册 Agent

curl -X POST https://api.simmer.markets/api/sdk/agents/register \
  -H "Content-Type: application/json" \
  -d '{"name": "my-agent", "description": "My trading agent"}'

保存返回的 api_key，后续所有请求都需要。

2. 发送 Claim 链接

将返回的 claim_url 发送给你的真人用户，完成验证后可解锁实盘 USDC 交易。

💡 提示：Claim 流程用于合规验证，确保真人监督 AI 交易。

3. 验证状态

export SIMMER_API_KEY="sk_live_..."
curl https://api.simmer.markets/api/sdk/agents/me \
  -H "Authorization: Bearer $SIMMER_API_KEY"

应返回 agent 信息、余额和交易权限。

📊 交易 Venues

Simmer ($SIM)

虚拟盘练习，默认 venue，无真实资金风险

Polymarket (USDC)

实盘交易，需 claim 后解锁，全球最大预测市场

Kalshi (USD)

美国合规市场，需 Pro plan，受 CFTC 监管

切换 Venue

# 在交易请求中指定 venue 参数
{
  "venue": "polymarket",  // 或 "simmer", "kalshi"
  "market": "btc-to-100k-by-2026",
  "side": "yes",
  "amount": 100
}

⚙️ 安全限额（默认）

Simmer 默认设置安全限额保护资金：

  • 单笔交易：$100
  • 每日限额：$500
  • 每日交易次数：50 笔

调整限额

需要联系 Simmer 团队或通过 claim 用户账户调整。建议：

  • 初期保持默认限额
  • 模拟盘测试 2-4 周后再提高
  • 根据策略表现逐步调整
⚠️ 风控建议：即使提高限额，也应设置熔断机制，连续亏损时自动暂停。

🔧 常用 API 端点

获取市场信息

curl "https://api.simmer.markets/api/sdk/markets?search=btc" \
  -H "Authorization: Bearer $SIMMER_API_KEY"

下单交易

curl -X POST https://api.simmer.markets/api/sdk/orders \
  -H "Authorization: Bearer $SIMMER_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "venue": "polymarket",
    "market": "btc-to-100k-by-2026",
    "side": "yes",
    "amount": 100,
    "type": "market"
  }'

查询持仓

curl https://api.simmer.markets/api/sdk/portfolio \
  -H "Authorization: Bearer $SIMMER_API_KEY"

查询交易历史

curl "https://api.simmer.markets/api/sdk/orders?limit=50" \
  -H "Authorization: Bearer $SIMMER_API_KEY"

⚠️ 注意事项

  • 私钥安全：Simmer 使用自托管钱包，私钥本地加密存储
  • Gas 费：Polymarket 交易需要 POL（Polygon 网络 Gas）
  • 结算延迟：市场结算依赖预言机，可能延迟数小时
  • API 限流：默认 100 请求/分钟，高频策略需申请提高
