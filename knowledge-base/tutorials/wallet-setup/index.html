👛 钱包配置指南

最后更新：2026-03-07 · 来源：Polymarket + Simmer 官方文档
安全 Metamask Polygon

📌 概述

Polymarket 运行在 Polygon 网络上，需要配置支持 Polygon 的钱包并持有 USDC.e（桥接 USDC）和少量 POL（Gas 费）。

⏱️ 预计时间：10-15 分钟

步骤 1：安装 MetaMask

浏览器扩展（推荐）

  1. 打开 Chrome/Edge/Firefox 浏览器
  2. 访问 metamask.io
  3. 点击 "Download" 安装扩展
  4. 创建新钱包或导入现有钱包

移动端

  • iOS：App Store 搜索 "MetaMask"
  • Android：Google Play 搜索 "MetaMask"

⚠️ 安全提示：助记词只能保存一次，请离线备份（纸笔记录），不要截图或存云端！

步骤 2：添加 Polygon 网络

MetaMask 默认是 Ethereum 主网，需要手动添加 Polygon：

自动添加（推荐）

  1. 访问 polygon.technology
  2. 点击 "Add Polygon to MetaMask"
  3. 在 MetaMask 弹窗中确认添加

手动添加

网络名称：Polygon Mainnet
RPC URL: https://polygon-rpc.com
链 ID: 137
货币符号：POL
区块浏览器：https://polygonscan.com

步骤 3：获取 USDC.e

Polymarket 使用 USDC.e（桥接 USDC），不是原生 USDC。

方式 A：从交易所提现

  1. 在币安/Bybit 购买 USDC
  2. 选择提现 → 网络选择 Polygon
  3. 输入你的 MetaMask 地址（以 0x 开头）
  4. 等待 5-10 分钟到账

方式 B：使用桥接

  1. 访问 bridge.polygon.technology
  2. 连接钱包
  3. 选择 Ethereum → Polygon
  4. 输入金额，确认桥接

💡 建议：首次转入$50-100 测试，确认流程后再转大额。

步骤 4：获取 POL（Gas 费）

Polygon 网络交易需要 POL 作为 Gas 费，每笔交易约$0.01-$0.10。

获取方式

  • 交易所提现：币安/Bybit 购买 POL，选择 Polygon 网络提现
  • 桥接：在 bridge.polygon.technology 从 ETH 桥接
  • 兑换：在 Polymarket 内用 USDC 兑换少量 POL

💰 建议持有量：5-10 POL（约$2-5），可用很久。

步骤 5：连接 Polymarket

  1. 访问 polymarket.com
  2. 点击右上角 "Connect"
  3. 选择 MetaMask
  4. 在 MetaMask 弹窗中确认连接
  5. 连接成功后显示钱包余额

步骤 6：配置 Simmer（可选）

如果使用 Simmer 进行自动化交易，需要配置外部钱包：

导出私钥

  1. MetaMask → 设置 → 安全与隐私
  2. 点击 "显示私钥"
  3. 输入密码，复制私钥（0x 开头的 64 位十六进制）

⚠️ 极度危险：私钥 = 钱包控制权！永远不要：

  • ❌ 发送给任何人
  • ❌ 上传到 GitHub
  • ❌ 存在云端或聊天软件
  • ✅ 只用于 Simmer 等可信服务的本地配置

配置 Simmer

export WALLET_PRIVATE_KEY="0x你的私钥"
export SIMMER_API_KEY="sk_live_..."

配置后运行：

notebooklm login  # 如果用到 NotebookLM 功能

✅ 验证配置

  • 打开 Polymarket，右上角显示余额
  • 尝试买入$1 测试（任何市场）
  • 确认 MetaMask 弹出签名请求
  • 交易成功后在 "Portfolio" 查看持仓

⚠️ 常见问题

Q: 转账后余额不显示？

A: 等待区块确认（5-10 分钟），刷新页面或切换网络再切回。

Q: Gas 费不足怎么办？

A: 需要 POL 代币。在 Polymarket 设置中可以兑换，或从交易所提现。

Q: 如何查看交易历史？

A: 访问 polygonscan.com，输入钱包地址查询。
