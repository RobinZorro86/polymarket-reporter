Simmer 自动交易 教程

📌 核心贡献

@edwordkaru 专注于 Simmer 自动交易教程和概率套利原理的普及，强调用旧手机 + OpenClaw 实现低成本自动化交易。

💡 核心理念：用 10,000 SIM 虚拟盘练习，胜率 70% 以上再进入实盘。

📱 旧手机自动交易方案

硬件要求

  • 闲置智能手机（Android/iOS 均可）
  • 稳定 WiFi 连接
  • 24 小时插电运行

软件安装

# 安装 OpenClaw
npm install -g openclaw

# 安装 Simmer 技能
npx clawhub@latest install simmer

🎯 可用技能

1. Weather Trader（天气交易）

  • 市场：NYC、Chicago、Seattle、Atlanta、Dallas、Miami 温度预测
  • 策略：基于 NOAA 数据，寻找错误定价
  • 频率：每日扫描，发现机会自动入场

2. Mert Sniper（临近到期狙击）

  • 市场：BTC/ETH/ SOL 等自选赛道
  • 策略：临近到期市场的 Conviction Trading
  • 频率：持续扫描，机会出现立即执行

📊 保守策略配置示例

# 策略参数
Entry threshold: 13%    # 低于 13% 买入
Exit threshold: 35%     # 高于 35% 卖出
Max position: $100      # 单笔最大仓位
Max trades/run: 5       # 每次运行最多 5 笔

# 支持城市
Locations: NYC, Chicago, Seattle, 
           Atlanta, Dallas, Miami

🎓 训练模式建议

  1. 第 1 阶段：用 10,000 $SIM 虚拟盘练习 1-2 周
  2. 第 2 阶段：记录每笔交易的决策逻辑
  3. 第 3 阶段：胜率稳定 70% 以上
  4. 第 4 阶段：开启小额实盘（$100/笔）

⚠️ 重要：先用虚拟交易累计数据，再进入正式交易。不要跳过训练阶段。

📐 概率套利核心原理

Polymarket 与传统套利不同：

  • 不能做空或使用杠杆，只能买入并管理退出
  • 寻找概率之和低于 100% 的组合（而非传统套利的>100%）
  • 单笔 YES + 单笔 NO = $1，这是概率基本定义
