📊 Polymarket API

市场数据 API

获取市场列表、价格、流动性等公开数据。

GET https://gamma-api.polymarket.com/events?limit=10&active=true
GET https://gamma-api.polymarket.com/markets/{marketId}

完整文档 →

CLOB API（订单簿）

限价订单簿接口，支持下单、撤单、查询持仓。

GET https://clob.polymarket.com/orderbook?token_id={tokenId}
POST https://clob.polymarket.com/order
GET https://clob.polymarket.com/balance?asset_type=collateral

认证：需要 API Key + 签名
CLOB 文档 →

🔮 Simmer API

Agent 管理

注册和管理 AI Agent，获取 API Key。

POST https://api.simmer.markets/api/sdk/agents/register 
{
  "name": "my-agent",
  "description": "Trading agent"
}
GET https://api.simmer.markets/api/sdk/agents/me

Headers: Authorization: Bearer sk_live_...

市场数据

GET https://api.simmer.markets/api/sdk/markets?q=btc&limit=10
GET https://api.simmer.markets/api/sdk/context/{marketId}

获取市场上下文、警告、持仓信息

交易执行

POST https://api.simmer.markets/api/sdk/trade 
{
  "market_id": "abc123",
  "side": "yes",
  "size": 10.0,
  "source": "sdk:strategy",
  "reasoning": "NOAA forecasts 35°F"
}

Simmer 完整文档 →

简报（Heartbeat）

一次性获取所有持仓、风险警报、推荐操作。

GET https://api.simmer.markets/api/sdk/briefing?since=1772800000

返回 Simmer、Polymarket、Kalshi 全 venue 数据
