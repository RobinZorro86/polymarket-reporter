📊 市场术语

预测市场 (Prediction Market)

参与者对事件结果进行交易的金融市场。价格反映市场对事件发生概率的共识。

YES / NO 份额

二元市场的两种合约。事件发生后，正确预测的份额价值$1.00，错误的归零。

市场价格 (Market Price)

当前交易价格，范围$0.00-$1.00。$0.60 表示市场认为有 60% 概率发生。

流动性 (Liquidity)

市场深度，决定大额订单的滑点。流动性越高，交易越容易成交。

Sprint 市场

短期快速结算的市场（5-15 分钟），如"下一个 5 分钟 BTC 涨还是跌"。

💰 交易术语

Maker

挂出限价单提供流动性的人。享受 0% 手续费。

Taker

吃单消耗流动性的人。支付 2% 手续费。

滑点 (Slippage)

实际成交价与预期价格的差异。流动性低时滑点较大。

限价单 (Limit Order)

设定特定价格等待成交的订单。可能不成交，但无滑点。

市价单 (Market Order)

以当前最优价格立即成交的订单。快速但有滑点风险。

🔗 区块链术语

Polygon

以太坊 Layer 2 扩展网络，Polymarket 运行在其上。Gas 费低廉（约$0.01-$0.10）。

USDC.e

桥接到 Polygon 的 USDC 稳定币。Polymarket 的交易货币。

POL

Polygon 网络的原生代币，用于支付 Gas 费。

Gas Fee

区块链交易手续费。Polygon 网络约$0.01-$0.10/笔。

智能合约 (Smart Contract)

自动执行的区块链程序。Polymarket 市场由智能合约管理。

🤖 自动化术语

OpenClaw

AI 助手执行框架，支持技能扩展和自动化工具调用。

Simmer

AI Agent 专用交易接口，支持 Polymarket 和 Kalshi，带安全限额。

Skill

OpenClaw 的可插拔功能模块，如交易策略、数据抓取等。

CLOB (Central Limit Order Book)

中央限价订单簿。Polymarket 的高频交易接口。
