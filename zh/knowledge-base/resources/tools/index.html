📌 核心工具

🦞 OpenClaw

AI 助手执行框架，支持技能扩展、自动化工具调用和多平台消息集成。

GitHub → · 文档 → 

npm install -g openclaw
openclaw onboard

📊 polymarket-cli

Polymarket 官方 CLI 工具，支持市场查询、订单管理、CLOB 数据访问。

GitHub → · 文档 → 

# 安装（Homebrew）
brew tap Polymarket/polymarket-cli
brew install polymarket

# 安装（脚本）
curl -sSL https://raw.githubusercontent.com/Polymarket/polymarket-cli/main/install.sh | sh

# 查询市场
polymarket markets list --limit 10
polymarket markets get will-trump-win-2024

# CLOB 数据
polymarket clob book TOKEN_ID
polymarket clob price TOKEN_ID --side buy

🔮 Simmer SDK

AI Agent 专用交易接口，支持 Polymarket 和 Kalshi，自托管钱包 + 安全限额。

官网 → · 文档 → 

# Python SDK
pip install simmer-sdk

# 注册 Agent
curl -X POST https://api.simmer.markets/api/sdk/agents/register \
  -H "Content-Type: application/json" \
  -d '{"name": "my-agent"}'

# 环境变量
export SIMMER_API_KEY="sk_live_..."
export WALLET_PRIVATE_KEY="0x..."
