## 📊 数据来源

### API 端点

**Polymarket API**:
```
GET /markets?category=weather
GET /markets/{market_id}/orderbook
GET /markets/{market_id}/trades
```

**NOAA API**:
```
GET /stations/{station_id}/observations/latest
GET /stations/{station_id}/records
```
