## Day 0：今天就开始（30 分钟）

### 第 1 步：搞懂 Polymarket 是什么（5 分钟）

Polymarket 是一个**预测市场**。

你可以在上面用真钱押注「某件事会不会发生」。

**关键机制**：

| 概念 | 含义 |
|------|------|
| **Yes 市场** | 押注「会发生」，价格 = 发生概率 |
| **No 市场** | 押注「不会发生」，价格 = 不发生概率 |
| **结算** | 判断正确得 $1，错误得 $0 |
| **价格** | 50¢ = 市场认为 50% 会发生 |

**举例**：
> 「2026 年比特币会突破 $100,000 吗？」
> 
> 当前价格：**65¢**
> 
> 含义：市场认为这件事有 **65%** 的概率发生
> 
> 你如果买 Yes，价格 $0.65，判断正确得 $1，收益 53.8%

### 第 4 步：理解「价格 = 概率」这个关键（5 分钟）

Polymarket 上，**市场价格就是市场共识概率**。

你的任务是：**判断市场价格是否偏离了真实概率**。

```
市场价格 40¢ = 市场认为 40% 会发生

如果我认为真实概率是 60%：
→ 市场价格被低估了
→ 买入 Yes 有正 EV
→ 值得考虑交易

如果我认为真实概率是 30%：
→ 市场价格被高估了
→ 买入 No 有正 EV
→ 值得考虑交易
```

**今天的任务**: 找到 1 个你判断概率和市场价格「有明显差距」的市场。

## Day 1：学会 EV 计算（20 分钟）

### 什么是 EV（期望值）？

EV 是判断一笔交易是否值得做的核心公式：

```
EV = (你的胜率 × 盈利) - (你的败率 × 亏损)
```

**Polymarket 简化版**：
```
EV = (P × $1) - ((1-P) × 投入金额)
```

其中 **P = 你的真实概率估计**

---

### 实际计算练习

**场景**：

- 市场：「2026 年 ETH 会突破 $5,000 吗？」
- 当前价格：**45¢**（市场认为 45% 概率发生）
- 你认为真实概率是 **60%**
- 你计划投入 **$10**

**计算**：
```
EV = (0.60 × $10) - (0.40 × $10)
   = $6 - $4
   = +$2
```

**EV 为正，说明这笔交易在数学上值得考虑。**

---

### 练习题

用下面这个市场练习 EV 计算：

- 市场：「下周 BTC 会创新高吗？」
- 价格：**55¢**
- 你认为概率：**65%**
- 计划投入：**$20**

```
EV = (_____ × _____) - (_____ × _____)
   = _____
```

**答案**：
```
EV = (0.65 × $20) - (0.35 × $20)
   = $13 - $7
   = +$6（正 EV）
```

## Day 2：学会估计真实概率 P（20 分钟）

EV 计算里最重要的变量是 **P（你的真实概率估计）**。

P 估计得准不准，直接决定 EV 正负。

### 估计 P 的三个方法

#### 方法 1：基准率法（最可靠）

找历史上类似事件的发生率作为基准。

**举例**：
> 「下周美联储会加息吗？」
> 
> 查历史：过去 10 年美联储加息次数 / 会议次数 ≈ 30%
> 
> 基准率 P ≈ 30%

#### 方法 2：多源验证法（最实用）

不要只信一个来源，至少查 **3 个不同来源**：

```
来源 1：Bloomberg / Reuters 新闻
来源 2：专业分析师预测
来源 3：其他预测市场价格
```

取中间值或保守值作为你的 P。

#### 方法 3：市场共识偏差法（最聪明）

看市场价格代表什么概率，然后判断市场是否「低估」或「高估」了。

```
市场价格 30¢ = 市场认为 30% 会发生

如果你认为市场错了，真实概率应该是 50%：
→ 市场低估了 20 个百分点
→ 买入 Yes 有显著正 EV
```

## Day 3：学会仓位控制（15 分钟）

EV 为正只说明这笔交易在数学上值得做，但下多少同样重要。

### Kelly 公式（简化版）

```
建议仓位 = 半 Kelly = (P - 1) / 赔率 / 2
```

**实用规则**（不需要背公式）：
- 单笔交易不超过总资金的 **2-5%**
- EV 越正，仓位可以略大
- 不确定时，**永远用更小的仓位**

### 仓位速查表

| 你的信心程度 | 建议仓位 |
|-------------|---------|
| EV 微弱为正（0-10%）| 总资金的 1-2% |
| EV 中等为正（10-20%）| 总资金的 2-3% |
| EV 显著为正（20%+）| 总资金的 3-5% |
| 不确定 | 0%（不交易）|

---

## 核心原则（背下来）

### 原则 1：EV 必须为正
> 每笔交易前必须过 EV 检查，没有例外。

### 原则 2：P 估计要有依据
> 不能靠直觉，要查数据、查历史、查多个来源。

### 原则 3：仓位永远要小
> 单笔不超过 5%，不确定就 1% 或 0%。

### 原则 4：记录每一天
> 不记录的复盘等于没有复盘。

### 原则 5：先验证，再加大
> 小额验证判断对了，再考虑加大仓位。

## 常见问题

**Q：我没有任何金融背景，能学会吗？**
能。关键不是背景，而是习惯：每笔交易前算 EV，每笔交易后记录复盘。

**Q：第一天就要真钱交易吗？**
建议小额真实交易（$1-5），因为真钱和模拟的心态完全不同，早点体验真实心态比学到更多东西更重要。

**Q：如果连续亏损怎么办？**
停下来复盘。你的 P 估计系统可能有问题，而不是因为「运气不好」。

**Q：学多久才能稳定赚钱？**
没有人能保证稳定赚钱。但如果你能保持「EV 为正的交易比例接近 100%」，长期来看期望收益为正是大概率的。
