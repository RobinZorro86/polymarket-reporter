🦞 OpenClaw 完整配置

最后更新：2026-03-07 · 来源：OpenClaw 官方文档
入门必备 环境配置 技能安装

📌 简介

OpenClaw 是 AI 助手执行框架，支持技能扩展、自动化工具调用和多平台消息集成。本教程从零开始配置完整的交易辅助系统。

⏱️ 预计时间：30-60 分钟

🔧 步骤 1：安装 OpenClaw

npm install -g openclaw
openclaw onboard

⚙️ 步骤 2：配置 Provider

编辑 ~/.openclaw/openclaw.json，添加模型 Provider：

{
  "models": {
    "providers": {
      "bailian": {
        "baseUrl": "https://coding.dashscope.aliyuncs.com/v1",
        "apiKey": "sk-your-api-key",
        "api": "openai-completions"
      }
    }
  }
}

📦 步骤 3：安装 Skills

# Simmer 交易技能
npx skills add joeseesun/anything-to-notebooklm -y

# 网页内容提取
npx skills add joeseesun/defuddle-skill -y

# 设计顾问
npx skills add joeseesun/qiaomu-design-advisor -y

# 学习网站生成
npx skills add joeseesun/knowledge-site-creator -y

🔑 步骤 4：配置 API Keys

⚠️ 安全提示：不要将 API keys 提交到 Git 仓库！使用 .env 文件或环境变量管理。

export SIMMER_API_KEY="sk_live_..."
export MEMOS_API_KEY="..."

✅ 验证安装

openclaw status
npx skills list
