🔮 Simmer 交易指南

最后更新：2026-03-07 · 来源：Simmer 官方文档
交易入门 虚拟盘 实盘

📌 什么是 Simmer

Simmer 是 AI Agent 专用的预测市场交易接口，支持 Polymarket 和 Kalshi 两个平台。提供自托管钱包、安全限额和智能上下文建议。

💰 起始资金：注册即送 10,000 $SIM（虚拟币），可用于练习交易

🚀 快速开始

1. 注册 Agent

curl -X POST https://api.simmer.markets/api/sdk/agents/register \
  -H "Content-Type: application/json" \
  -d '{"name": "my-agent", "description": "My trading agent"}'

保存返回的 api_key，后续所有请求都需要。

2. 发送 Claim 链接

将返回的 claim_url 发送给你的真人用户，完成验证后可解锁实盘 USDC 交易。

3. 验证状态

export SIMMER_API_KEY="sk_live_..."
curl https://api.simmer.markets/api/sdk/agents/me \
  -H "Authorization: Bearer $SIMMER_API_KEY"

📊 交易 venues

  • Simmer（$SIM） - 虚拟盘练习，默认 venue
  • Polymarket（USDC） - 实盘交易，需 claim 后解锁
  • Kalshi（USD） - 美国合规市场，需 Pro plan

⚙️ 安全限额（默认）

  • 单笔交易：$100
  • 每日限额：$500
  • 每日交易次数：50 笔

📚 推荐 Skills

  • polymarket-weather-trader - 天气预测套利
  • polymarket-copytrading - 追踪高胜率钱包
  • polymarket-mert-sniper - 临近到期交易
