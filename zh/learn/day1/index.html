🎯 你今天要搞懂什么

  • 市场价格本质上是概率表达，例如 0.63 约等于 63% 概率
  • Yes/No 合约不是股票，它的上限由市场结算决定
  • 价格波动来自新信息、流动性变化、群体反应和时间衰减
  • 你不是在猜“会不会发生”，而是在判断“当前定价是否合理”

🧠 核心公式与直觉

  • 隐含概率 ≈ 当前价格
  • 预期 edge = 你估计的真实概率 − 市场给出的概率
  • 判断重点不是事件会不会发生，而是市场有没有错价
学习建议：把每个市场都先当成一个“概率表达器”，不要急着把它当成投机场。
