第7天：实战演练与复盘

最后一天不是学新概念，而是把前 6 天串起来，形成你自己的执行框架。

完整闭环

  • 发现机会：新闻 / X / CLI 数据 / KOL
  • 形成假设：哪里错价、错多少、为什么错
  • 执行：仓位、时机、退出
  • 复盘：对在哪里、错在哪里、流程哪里能自动化

复盘模板

  • 市场名称
  • 入场理由
  • 是否符合策略规则
  • 退出原因
  • 结果与改进点

今日任务

  • 选一个近期市场做纸上演练
  • 按模板写完整复盘
  • 把复盘结果归档到你的知识体系里

使用建议：先读完本页，再进入对应策略页、资源页和日报案例，形成“概念 → 工具 → 策略 → 复盘”的闭环。
