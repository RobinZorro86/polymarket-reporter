🎯 完整闭环

  • 发现机会：新闻 / X / CLI 数据 / KOL
  • 形成假设：哪里错价、错多少、为什么错
  • 执行：仓位、时机、退出
  • 复盘：对在哪里、错在哪里、流程哪里能自动化

🧠 复盘模板

  • 市场名称
  • 入场理由
  • 是否符合策略规则
  • 退出原因
  • 结果与改进点
复盘纪律：每笔交易无论盈亏都应该带来具体收获。

✅ 今日任务

  • 选一个近期市场做纸上演练
  • 按模板写完整复盘
  • 把复盘结果归档到你的知识体系里
